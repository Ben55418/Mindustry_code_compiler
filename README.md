# Mindustry_code_compiler
code compiler for the Mindustry game

Python syntax, any lines not recognized will be added raw

## Currently Supporting:
```
if
elif
else
while loop

max()
min()
abs()

^
**
%
*
//
/
+
-
<<

<=
<
>=
<>
!=
===
==

referencing cells (reading) by name and brackets: cell1[4 + x]
```
